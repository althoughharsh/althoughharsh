## 🏆 Featured Projects

### 🌊 OceanLens – Conversational AI for Marine Data

**An intelligent chatbot transforming ocean data accessibility**

- **Architecture:** RAG Pipeline + LLaMA-2-7B + Real-time React Dashboard
- **Data Scale:** 3TB+ ARGO ocean dataset with multi-dimensional queries
- **Impact:** Enables climate-related decision support for the $8B+ global fishery sector
- **Team:** Collaborated with 6 interdisciplinary members on production deployment
- **Technologies:** React.js • Python • LLaMA-2 • Vector Databases • AWS

**Key Achievement:** Built intelligent marine data exploration tool improving research accessibility and enabling coastal communities in climate decision-making.
